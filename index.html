About Me

Software Engineer

ผมมีความมุ่งมั่นที่จะเป็น Software Engineer เพราะตลอดเวลาที่ผ่านมาผมได้ลงมือสร้างผลงานจริงทั้งแอปพลิเคชันอย่างง่าย My School Planner รวมถึงการทำ portfolio website นี้ที่ช่วยพัฒนาทักษะการเขียนโค้ด การแก้ปัญหา และการคิดสร้างสรรค์ ประสบการณ์เหล่านี้สะท้อนให้เห็นถึงความตั้งใจในการเรียนรู้และพัฒนาตัวผมเองอย่างต่อเนื่อง

สำหรับเป้าหมายในอนาคตของผม ผมจะต่อยอดทักษะทั้งด้านการพัฒนา Software เต็มรูปแบบ การทำงานกับ Data และ AI เพื่อสร้างเทคโนโลยีหรือนวัตกรรมที่มีประโยชน์ต่อสังคมรวมถึงยกระดับชีวิตของผู้คนทั่วไป ทั้งหมดนี้ทำให้ผมอยากเป็น Software Engineer นั่นเองครับ

My skills

Web Development

ผมมีทักษะในการสร้างเว็บไซต์พื้นฐานโดยใช้ HTML, CSS และ JavaScript ผ่านการทำ Portfolio Website ที่แสดงถึงความสามารถในการจัดโครงสร้างหน้าเว็บ ออกแบบการแสดงผล และเพิ่มฟังก์ชันการทำงานเชิงโต้ตอบ ซึ่งถือเป็นพื้นฐานสำคัญในการต่อยอดสู่การพัฒนาเว็บขั้นสูงหรือ Full-stack Development นั่นเองครับ

Mobile Development

ผมเคยสร้างแอป My School Planner ด้วย Google AppSheet ซึ่งสะท้อนความเข้าใจการออกแบบระบบ การจัดการข้อมูล และการสร้างฟังก์ชันการทำงานให้ตอบโจทย์ผู้ใช้ ถึงแม้จะเป็น Low-code Platform แต่ก็ช่วยให้ผมมีทักษะการคิดเชิงโครงสร้างและการพัฒนาแอปพลิเคชันจริงได้ครับ

Data analyse

ผมมีประสบการณ์วิเคราะห์ข้อมูล เช่น การหาปัจจัยที่มีผลต่อเงินเดือนแรกเข้า ซึ่งแสดงถึงการใช้เหตุผลเชิงตรรกะ การจัดการข้อมูล และการตีความเชิงสถิติ ทักษะนี้ทำให้ผมสามารถพัฒนาต่อไปสู่สาย Data Analytics, Machine Learning หรือ AI Engineering ได้ครับ

My Projects

My School Planner

โปรเจกต์ My School Planner เกิดจากปัญหาการจัดการการบ้าน ตารางสอบ และกิจกรรมที่ทำให้ผมมักหลงลืมงานบางส่วน จึงสร้างแอปต้นแบบด้วย Google AppSheet เชื่อมต่อฐานข้อมูลจาก Google Sheet และออกแบบฟีเจอร์ Homework, Daily Task, Calendar, Timetable, Project และ Note ผลลัพธ์คือช่วยให้ผมจัดการเวลาและภาระงานได้เป็นระบบมากขึ้นครับ

Portfolio Website

โปรเจกต์ Portfolio Website ผมสร้างเพื่อแสดงผลงานและข้อมูลในการสมัครมหาวิทยาลัย โดยใช้ HTML, CSS และ JavaScript เริ่มจากการออกแบบโครงร่าง กำหนดหน้าแนะนำตัว ทักษะ และผลงาน จากนั้นพัฒนาและทดสอบให้รองรับทุกอุปกรณ์ เว็บไซต์นี้ทำให้ผมมีสามารถนำเสนอผลงานและศักยภาพด้านเทคโนโลยีของผมได้อย่างชัดเจนครับ

Analysis of Factors Affecting Starting Salary

โปรเจกต์ Analysis of Factors Affecting Starting Salary มาจากความสนใจของผมในการวิเคราะห์ข้อมูล โดยใช้ Python บน Google Colab กับข้อมูลจาก Kaggle ผ่านขั้นตอนการทำความสะอาด วิเคราะห์เชิงสถิติและ Visualization ก่อนสร้างโมเดล Linear Regression เพื่อหาปัจจัยที่มีผลต่อเงินเดือนแรกเข้า ผลการวิเคราะห์ทำให้ผมได้เห็นอิทธิพลของระดับการศึกษา ประสบการณ์ และเพศ และเป็นแรงบันดาลใจให้ผมพัฒนาทักษะด้าน Data และ AI ต่อไปครับ
